✈️ AeroDesk – Flight Booking Web Application
📌 Project Overview

AeroDesk is a simple and user-friendly flight booking web application that allows users to register, log in, search for flights, book trips, and manage their travel history.
The project focuses on creating a clean UI and smooth user experience using core web technologies.

This project demonstrates fundamental front-end web development skills including page navigation, form handling, and interface design.

🚀 Features

👤 User Registration and Login

🏠 Home Dashboard

✈️ Flight Booking System

📅 Upcoming Trips View

✔️ Completed Trips Section

❌ Cancelled Trips Tracking

📄 Booking Confirmation Page

👤 User Profile Page

🎨 Responsive UI with shared styling
